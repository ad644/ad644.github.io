Hugo static site generator - Installation under Windows® and basic usage

Apr 23, 2021
3 min read
Windows Web HTML

Static site generators (SSG) has gain lot of attention recently because of simple process of creating and managing websites. Websites created by static site generators are blazing fast and usually loads within seconds! No matter it’s a personal blog or a professional website, SSG got it covered.

A static site generator is generally a set of tools to build static HTML websites based on templates and raw data. Nowadays it has become a popular alternative to traditional CMS (i.e. Wordpress) because of scale, performance and better security.

Popular static site generators

  • Jekyll
  • Hugo
  • Gatsby
  • Next.js

I personally prefer Hugo as SSG. It is written in Go language, has seen lot of improvements in features and performance over the years and most importantly it is open source (licensed under Apache License 2.0). It is easy to deploy and don’t require much expertise to configure according to needs.

This article will explain how to install and run Hugo under the most commonly used operating system, Microsoft Windows®.

Installation

  1. Download latest version of Hugo for Windows from here.
  2. Extract the zip archive and copy the executable file (Hugo) from that extracted folder to any other place as you like. (i.e. C:\Hugo\bin)
  3. You may want to add that location to your system path, so that you can call the ‘hugo’ command from anywhere.
  4. Now you can check the installation by opening command prompt (CMD) and issuing following command:
hugo version

Basic Usage

  1. Go to the location where you want to create your static website.
  2. Open CMD and issue following command to create the folder for you new website:
hugo new site new_site

Here, new_site is an example, you can give it any name as per your need.

  1. Now you need to download a Hugo theme and put it under “themes” directory (create it if not there) of the newly created “new_site” folder. You can get a theme from here.

  2. Then edit the config.toml file located inside the “new_site” folder (better to use Notepad++ or any other code editor) and add following line in that file-

theme = "theme_name"

You can also edit other parameters in that file, like- title of your website, base URL and so on.

  1. After basic configuration, you can run a local server with below command:
hugo server -D

  1. Open a web browser and go to localhost:1313 from address bar to see the website up and running. if you make any changes, you can see those live.

  2. Following command will generate static HTML website contents for hosting:

hugo -D

output will be under “public” directory by default.

However, if you want to learn more about configuration and management of Hugo then don’t forget to check Hugo docs.
